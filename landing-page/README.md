El sitio web público principal de **Initeck**. Presenta los servicios, áreas tecnológicas y propuesta de valor de la empresa con un diseño moderno, responsive e interactivo. Es el primer punto de contacto para clientes y conductores potenciales.

---

## 📁 Estructura del Proyecto

```
landing-page/
├── src/
│   └── assets/         # Imágenes, logos e íconos
├── dist/               # Build de producción
├── node_modules/
└── README.md
```

---

## ✨ Secciones del Sitio

| Sección | Descripción |
|---|---|
| 🏠 **Hero** | Presentación principal con llamada a la acción |
| 🔧 **Áreas Tecnológicas** | Servicios y capacidades tecnológicas de Initeck |
| 📱 **Productos** | Showcase del ecosistema (Tracker App, Safar) |
| 📞 **Contacto** | Formulario y datos de contacto |

---

## 🛠️ Tech Stack

| Tecnología | Uso |
|---|---|
| **React** | Framework de UI |
| **Vite** | Bundler y servidor de desarrollo |
| **Vanilla CSS** | Estilos premium con glassmorphism y animaciones |
| **Google Fonts** | Tipografía moderna (Inter / Outfit) |

---

## 🎨 Características de Diseño

- **Dark mode** por defecto con paleta de colores curada
- **Glassmorphism** en el header y componentes destacados
- **Micro-animaciones** en elementos interactivos y hover effects
- **Diseño responsive** optimizado para móvil y desktop
- **Gradientes** suaves en secciones clave

---

## ⚙️ Instalación y Desarrollo

```bash
cd landing-page
npm install
npm run dev       # Servidor de desarrollo → http://localhost:5173
npm run build     # Build de producción → dist/
```

---

## 🚀 Despliegue

1. Ejecuta `npm run build`
2. Sube el contenido de `dist/` al servidor web
3. Configura el servidor para servir `index.html` en todas las rutas
